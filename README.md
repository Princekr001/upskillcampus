# 🏦 Banking Information System

## 📝 Project Description
A console-based Banking Information System developed using Core Java. The project allows users to create accounts, login securely, deposit and withdraw money, transfer funds, and view a detailed transaction history. It serves as a comprehensive prototype demonstrating fundamental object-oriented programming concepts and system architecture.

## ✨ Features
- **User Registration:** Create new accounts with unique auto-generated 10-digit account numbers.
- **Login Authentication:** Secure access to personal dashboards using an account number and password.
- **Deposit & Withdraw Money:** Manage account balances with built-in mathematical validation.
- **Fund Transfer:** Send money directly to other registered users in the system.
- **Transaction History / Account Statement:** Maintain a timestamped, detailed record of all account activities.
- **Robust Error Handling:** Input validation to prevent crashes, negative deposits, and overdrawing funds.

## 💻 Technologies Used
- **Language:** Core Java
- **Core Concepts:** Object-Oriented Programming (OOP)
- **Data Structures:** Java Collections Framework (`ArrayList`, `HashMap`)
- **Other:** Exception Handling, Java Time API

## 📂 Project Structure
- `Main.java` - The entry point and interactive console user interface.
- `User.java` - The data model representing a bank account and its owner.
- `BankService.java` - The core business logic handling authentication, user persistence, and financial operations.
- `Transaction.java` - The data model for tracking individual transaction timestamps and receipts.

## 🚀 Future Enhancements
